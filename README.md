# Paper-Mate - your personal assistant for exploring arXiv papers

### Start Paper Mate

1. clone the repository with ```git clone https://github.com/mechark/paper-mate.git```
2. Run ```cd paper-mate```
3. Make sure your  Docker Engine is running and type ```docker compose up```
4. Run ```docker exec -it ollama ollama pull mistral:7b-instruct-q4_0``` to download the model
